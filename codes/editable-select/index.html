jQuery Editable Select Plugin Examples

Basic example

It trasforms this
Into this

Code

$('#basic').editableSelect();

Effects

Default
Slide
Fade

Code

$('#default').editableSelect({ effects: 'default' });
$('#slide').editableSelect({ effects: 'slide' });
$('#fade').editableSelect({ effects: 'fade' });

No filter

Default

Code

$('#filter').editableSelect({ filter: false });

HTML support

Code

$('#html').editableSelect();

onSelect

  Selected:  

Code

$('#onselect')
    .editableSelect()
    .on('select.editable-select', function (e, li) {
        $('#afterSelect').html(
            li.val() + '. ' + li.text()
        );
    });
